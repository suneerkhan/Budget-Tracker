# 💰 Budget Tracker

A clean, responsive personal finance web app with a glassmorphism dark UI — built as part of a 10-day web development workshop.

## 🔗 Live Demo
[View Live →](https://suneerkhan.github.io/budget-tracker)

---

## ✨ Features

- **Add, edit & delete** income and expense transactions
- **Auto balance calculation** updates in real time
- **Category tags** — Salary, Food, Transport, Bills, and more
- **Monthly bar chart** — 6-month income vs expense overview
- **Search bar** — filter transactions by name or category
- **Date filter** — view All, This Week, or This Month
- **Currency switcher** — supports ₹, $, and €
- **Local storage** — data persists across sessions, no backend needed
- **Glassmorphism UI** — dark theme with frosted glass cards and gradient accents
- **Fully responsive** — works on mobile and desktop

---

## 🛠️ Built With

- HTML5
- CSS3 (Glassmorphism, Flexbox, Grid)
- JavaScript (Vanilla)
- Chart.js
